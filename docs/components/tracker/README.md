# Job Tracker Component

**Navigation ID:** `tracker`  
**Label:** Job Tracker  
**Section:** APPLY

## Overview

Comprehensive job application tracking system with Notion-like interface.

## Component Location

- Main Component: `apps/web/src/components/JobTracker.tsx`
- Related Components: `apps/web/src/components/jobs/`

## Features

- Multiple view modes (Table, Card, Kanban)
- Job application management
- Interview tracking
- Salary and offer tracking
- Company insights
- Referral tracking
- Notes and reminders
- Export functionality (CSV, XLSX)
- Application status pipeline
- Filtering and sorting
