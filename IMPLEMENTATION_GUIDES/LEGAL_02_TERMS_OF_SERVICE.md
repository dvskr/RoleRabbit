## Lawyer Review Checklist

Provide your lawyer with:
- ⬜ This Terms template
- ⬜ Your business model description

- ⬜ Target markets (countries)
- ⬜ List of third-party services used

- ⬜ Do we need arbitration clause?
- ⬜ Do we need class action waiver?
- ⬜ What jurisdiction should govern?

- ⬜ Do we need export control provisions?

## Version Control

Track Terms changes:

```javascript
// Prisma schema
model TermsVersion {
  id        String   @id @default(cuid())
  version   String   // "1.0", "1.1", etc.
  content   String   // Full Terms text
  changes   String   // Summary of changes
  effectiveDate DateTime
  createdAt DateTime @default(now())

  @@map("terms_versions")
}

// When updating Terms
await prisma.termsVersion.create({
  data: {
    version: '1.1',
    content: newTermsContent,
    changes: 'Added arbitration clause, updated refund policy',
    effectiveDate: new Date('2024-02-01')
  }
});

// Notify users of changes
await sendEmailToAllUsers({
  subject: 'Updated Terms of Service',
  body: 'We have updated our Terms. Review changes: https://rolerabbit.com/terms'
});
```

## Common Mistakes to Avoid

1. ❌ Using Terms without lawyer review
2. ❌ Copy-pasting from other companies

4. ❌ Not getting user acceptance
5. ❌ Not tracking acceptance in database
6. ❌ Not notifying users of changes
7. ❌ Overreaching liability limitations (may be unenforceable)
8. ❌ Not considering jurisdiction differences
