# Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels (used in browser toolbar)
- `icon32.png` - 32x32 pixels (used in context menus)
- `icon48.png` - 48x48 pixels (used in extension management page)
- `icon128.png` - 128x128 pixels (used in Chrome Web Store)

## Creating Icons

You can create these icons using any image editor:
1. Start with a 512x512 logo
2. Create versions at 16x16, 32x32, 48x48, and 128x128
3. Save as PNG files
4. Use a transparent background

## Icon Design

Recommended design:
- Use the RoleReady logo (purple gradient: #667eea to #764ba2)
- Simple, recognizable symbol
- Works well at small sizes
- High contrast for visibility

For now, you can use placeholder icons or generate them using an online tool.
